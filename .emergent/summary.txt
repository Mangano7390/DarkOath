<product_requirements>
The goal is to create Secretus Regnum, a real-time multiplayer web app inspired by Secret Hitler, with a medieval theme. The MVP targets 5 players (3 Loyal, 1 Conjuré, 1 Usurpateur). Key rules include tracking 6 Conjuré vs 5 Loyal decrees for victory, Usurpateur victory if elected Chambellan after 3 Conjuré decrees, and specific powers at decree thresholds (Investigation at 2 Conjurés, single Execution at 4 Conjurés). The MVP omits veto, uses code-based rooms, optional bots, and FR/EN localization (FR default). Core pages include Landing, Lobby (), Game Table (), and End screen. Anonymous login is required. The stack is React/Tailwind (frontend), FastAPI (backend), with in-memory state for MVP (preparing for Postgres/Redis). WebSocket for real-time was intended, but polling is currently used.
</product_requirements>
